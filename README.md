# AI/ML Engineering Internship Tasks

This repository contains completed internship tasks for the DevelopersHub Corporation AI/ML Engineering Internship Program.

The projects focus on:
- Data Analysis and Visualization
- Machine Learning
- Regression and Classification
- Prompt Engineering
- Conversational AI
- Model Evaluation

---

# Completed Tasks

## Task 1 — Exploratory Data Analysis on Iris Dataset
- Data loading and inspection
- Statistical analysis
- Scatter plots, histograms, and boxplots
- Exploratory Data Analysis (EDA)

## Task 2 — Stock Price Prediction
- Historical stock market analysis
- Linear Regression model
- Short-term stock price prediction
- Actual vs predicted visualization

## Task 3 — Heart Disease Prediction
- Medical dataset preprocessing
- Binary classification using Logistic Regression
- ROC curve and confusion matrix evaluation
- Feature importance analysis

## Task 4 — General Health Query Chatbot
- Prompt engineering with LLMs
- Safety-aware conversational AI
- Hugging Face Inference API integration
- Context-aware chatbot responses

## Task 6 — House Price Prediction
- Regression modeling
- Feature scaling and encoding
- MAE and RMSE evaluation
- House price prediction visualization
