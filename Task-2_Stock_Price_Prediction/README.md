# Task 2: Stock Price Prediction

## Objective
The goal of this project is to predict the next day's stock closing price using historical stock market data.

---

## Dataset
- Source: Yahoo Finance
- Library Used: yfinance
- Stock Selected: Apple (AAPL)

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- yfinance

---

## Features Used
- Open
- High
- Low
- Volume

Target Variable:
- Close Price

---

## Machine Learning Model
- Linear Regression

---

## Project Workflow
1. Fetch stock data
2. Data preprocessing
3. Feature selection
4. Train-test split
5. Model training
6. Prediction
7. Visualization of actual vs predicted prices

---

## Key Insights
- The model can capture short-term price trends reasonably well.
- Historical market indicators help predict closing prices.
- Visualization helps compare prediction accuracy.
