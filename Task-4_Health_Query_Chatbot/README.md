# Task 4 - General Health Query Chatbot

## Objective
The objective of this project is to build a simple AI-powered chatbot that answers general health-related questions using prompt engineering and a Large Language Model (LLM).

## Features
- Answers general health questions
- Friendly conversational responses
- Prompt engineering for better communication
- Safety filtering for harmful queries
- Simple command-line chatbot interface

## Technologies Used
- Python
- OpenAI API
- Prompt Engineering
- dotenv

## Example Questions
- What causes sore throat?
- Is paracetamol safe for children?
- What are symptoms of flu?
- How to reduce stress?

## Files Included
- chatbot.py → Main chatbot script
- requirements.txt → Required Python libraries
- Task4_Health_Chatbot.ipynb → Jupyter Notebook version
- README.md → Project documentation

## How to Run

### Install Dependencies
```bash
pip install -r requirements.txt
```

### Run the Chatbot
```bash
python chatbot.py
```

## Safety Features
The chatbot avoids:
- dangerous medical advice
- self-harm related discussions
- harmful recommendations

## Conclusion
This project demonstrates the use of prompt engineering and AI APIs to create a safe and user-friendly health assistant chatbot.
