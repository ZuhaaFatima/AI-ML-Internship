# Task 4: General Health Query Chatbot

## Objective
Build a chatbot that answers general health-related questions using a Large Language Model (LLM).

---

## Technologies Used
- Python
- Hugging Face Inference API
- Mistral-7B-Instruct
- Prompt Engineering

---

## Features
- Accepts user health questions
- Generates friendly and clear responses
- Uses prompt engineering for safer conversations
- Includes safety filtering for harmful medical advice

---

## Example Queries
- What causes a sore throat?
- Is paracetamol safe for children?
- How can I improve sleep quality?

---

## Safety Features
- Avoids diagnosis and emergency medical advice
- Encourages consulting healthcare professionals
- Blocks dangerous or unsafe medical requests

---

## Key Learning Outcomes
- Prompt engineering
- LLM API integration
- Conversational AI basics
- AI safety handling
